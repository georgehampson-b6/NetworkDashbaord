Setting Up iPerf Batch Files and Python Scripts for Running and Plotting

1. Install Python - First, you'll need to install Python:
	a. Download Python: Go to the official Python website and download the latest version of Python for Windows.
	b. Run the Installer: Open the downloaded file and run the installer. Make sure to check the box that says "Add Python to PATH" before clicking "Install Now."

2. Install nmap - Then you'll have to install nmap:
	a. Use this url: https://nmap.org/download#windows
	b. Use the latest stable release self-installer and follow all the instructions to download to your computer

2. Install Required Python Packages - You will need to install some Python packages which your script depends on:
	All of these packages are in a file titled requirements.txt
	To install the packages, just double click on the batch file: download_requirements.bat

3. Run the Batch File
	Double _run_Internet_Testing.bat to open a webpage encompassing various methods of internet testing

4. Move the Shortcut to wherever you want for convenience

5. Troubleshooting
	If you encounter any errors related to missing packages, make sure they are installed correctly via pip
